# Tucil2_13524071_13524109

## Deskripsi
Program C# untuk foundation voxelization objek 3D berbasis file `.obj` menggunakan struktur octree.

## Requirement
- .NET SDK 8.0 atau versi yang kompatibel

## Struktur Folder
- `src/` : source code
- `bin/` : executable file
- `test/` : file pengujian
- `doc/` : laporan

## Cara Build
Masuk ke folder `src`, lalu jalankan:

```bash
dotnet build
```

## Cara Menjalankan
Masuk ke folder `src`, lalu jalankan:

```bash
dotnet run -- <input.obj> <maxDepth>
```
